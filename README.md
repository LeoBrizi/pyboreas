# boreas-devkit
Devkit for the Boreas autonomous driving dataset.

## Installation

You need `pylgmath` and `pysteam` to be installed before proceeding:

1. Install [`pylgmath`](https://github.com/utiasASRL/lgmath/tree/feature/python3#install-using-pip)
2. Install [`pysteam`](https://github.com/utiasASRL/steam/tree/feature/python3#install-using-pip)


Installing the Boreas Devkit:
```
pip install -e boreas-devkit
```
